Display

A propriedade display é uma das mais fundamentais no CSS. Ela define como um elemento HTML é exibido na página, controlando o tipo de "caixa" de renderização que ele gera. Isso impacta diretamente como o elemento se comporta no fluxo do documento e como ele interage com outros elementos ao seu redor. Entender display é crucial para construir layouts e organizar o conteúdo de forma eficaz.

O que faz

Controla o tipo de caixa de renderização de um elemento, determinando seu fluxo, alinhamento, e como ele ocupa espaço no layout.

Sintaxe e Valores Comuns


          display: valor;

A propriedade display aceita uma ampla gama de valores, mas os mais utilizados e essenciais para a maioria dos layouts são:

  • block: O elemento se comporta como um bloco. Ele ocupa toda a largura disponível por padrão e sempre começa em uma nova linha. Exemplos de elementos HTML que são block por padrão incluem div, p, h1 a h6, ul, li.
  • inline: O elemento se comporta como um elemento em linha. Ele ocupa apenas o espaço necessário para seu conteúdo e não quebra a linha, permitindo que outros elementos inline ou inline-block fiquem ao seu lado. Exemplos de elementos HTML que são inline por padrão incluem span, a, strong, em, img. Importante: Você não pode definir width, height, margin-top, margin-bottom ou padding-top, padding-bottom para elementos inline.
  • inline-block: Este valor é uma combinação dos dois. O elemento se comporta como inline no fluxo (fica lado a lado com outros elementos), mas você pode definir width, height, margin e padding (incluindo os verticais) como faria com um elemento block. É extremamente útil para alinhar blocos de conteúdo horizontalmente.
  • flex: Transforma o elemento em um contêiner flexível (flex container). Seus filhos diretos se tornam itens flexíveis (flex items) e podem ser organizados de forma flexível usando as propriedades do Flexbox (como justify-content, align-items, flex-direction). Ideal para layouts unidimensionais (linha ou coluna).
  • grid: Transforma o elemento em um contêiner de grade (grid container). Seus filhos diretos se tornam itens de grade (grid items) e podem ser organizados em um layout complexo de duas dimensões (linhas e colunas) usando as propriedades do CSS Grid (como grid-template-columns, grid-template-rows, gap).
  • none: O elemento e todos os seus filhos são completamente removidos do fluxo do documento, como se nunca tivessem existido. Ele não ocupa nenhum espaço na página e não é visível. Diferente de visibility: hidden; que oculta o elemento mas mantém seu espaço.
